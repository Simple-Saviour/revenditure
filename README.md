# REVENDITURE

It is a budget tracker application that helps 250+ people handle their budgets more efficiently.<br>
It has a voice assistance feature that records your voice and adds the details accordingly.<br>
The application uses doughnuts to represent the percentage of expenditure and revenue of respective fields.
